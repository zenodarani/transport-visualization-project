Metadata of the columns

  • REPORTER : three-digit code of the country of origin of the connection.
  • Reporter Country : textual name of the reporter country.
  • FLOW : code representing whether the measurement was an importation or an exportation. (1 - Imports, 2 - Exports)
  • Flow : whether it was an importation or exportation.
  • PARTNER : three-digit code of the partner country.
  • Year: the year when the measurements were taken (2008-2021).
  • Value : the actual value of the trade in millions of USD.
